css 란?

css는 html 문서의 스타일을 지정한는데 사용하는 언어입니다. css는 html요소가 어떻게 표시되어야 하는지 설명합니다.

  • css는 cascading style Sheet 의 약자입니다.
  • css 작성은 태그에서 1) style 속성 사용하거나 또는 2)style 태그에 작성합니다.
  • css는 파일로 작성해서 여러 웹페이지(html파일)에 적용되게 할 수 있습니다.
  • style 태그에서는 특정요소에 적용할 선택자(selector) 필요합니다 참고로 style 태그는 head안에 작성합니다.
  • css를 저장하는 파일 확장자는 .css 입니다.
  • 확장자 css는 별도로 외부파일에 css를 작성할 때 사용합니다.
  • css프로퍼티는 글꼴 크기, 색상,배경이미지, 테두리 등등이 있습니다.
